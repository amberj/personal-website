Install Librechat on Mac

Install and Setup

  • Install Docker Desktop for Mac
  • Install Librechat inside Docker
  • Visit http://localhost:3080/ to access Librechat.
  • Enable Openrouter inside Librechat.
    • The librechat.yaml file should be placed in the root of the project where the .env file is located. Also use an override file with Docker.
    • Restarting container didnt work for me, so I destroyed the container (using Docker Desktop GUI app on Mac) and recreated it again.

Optional

  • To enable "web search", you need to add three API keys: Read more
  • To enable memory features, you need to add the memory configuration to your librechat.yaml file. Read more here.
